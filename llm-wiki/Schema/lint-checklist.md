# Lint Checklist

Use this checklist before every meaningful commit.

- Source notes are under `Raw/Sources/`.
- Compiled notes are under `Wiki/`.
- Compiled notes use one allowed tag only: `topic`, `concept`, `entity`, `project`, `log`, `method`, `equation`, `comparison`, `figure`, or `lecture-note`.
- Every compiled note has at least one valid source link.
- `source_count` equals the number of `sources`.
- Source links resolve to existing files in `Raw/Sources/`.
- Source notes marked processed have compiled Wiki coverage.
- Claims in compiled notes are traceable to source material.
- No invented citations or unsupported claims appear in compiled notes.
- Notes with many Raw sources should be reviewed for evidence precision.
