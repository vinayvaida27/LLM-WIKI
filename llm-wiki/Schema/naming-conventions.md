# Naming Conventions

Use predictable names so humans and agents can find notes quickly.

## File Names

- Use clear, descriptive names for note purpose.
- Prefer searchable words over abbreviations.
- Avoid placeholder names like `new.md`, `notes1.md`, or `misc.md`.
- Keep one topic per compiled note when possible.

## Folder Locations

- Source material: `Raw/Sources/`
- Topic hubs: `Wiki/Topics/`
- Concepts: `Wiki/Concepts/`
- Entities: `Wiki/Entities/`
- Projects: `Wiki/Projects/`
- Logs: `Wiki/Logs/`
