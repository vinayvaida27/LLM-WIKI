# LLM Wiki Ingest

Use this skill when new source material is added.

## Steps

1. Add cleaned source notes to `Raw/Sources/`.
2. Search existing compiled notes first to avoid duplicates.
3. Create or update focused notes under `Wiki/`.
4. Add source links in `sources` frontmatter.
5. Set `source_count` to match linked sources.
6. Keep notes short, reusable, and source-traceable.
7. Run build, lint, and source checks before commit.
