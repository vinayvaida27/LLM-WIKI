# LLM Wiki Maintain

Use this skill for routine maintenance and release hygiene.

## Steps

1. Rebuild indexes and catalog artifacts after content changes.
2. Refresh source coverage after ingest updates.
3. Run doctor, build, lint, and source checks before commit.
4. Keep generated artifacts committed when required by tutorial step.
5. Preserve traceability between compiled notes and source notes.
6. Reject unsupported claims and invented citations.
