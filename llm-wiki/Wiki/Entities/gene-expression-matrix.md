# Gene Expression Matrix

## Definition
A cells-by-genes (or samples-by-genes) matrix of RNA quantification values. The fundamental input data type for transcriptomics clustering and single-cell analysis.

## Why It Matters
Gene Expression Matrix is a canonical MLCB graph node. Keeping it as a source-backed Wiki page prevents Obsidian from creating a phantom node for `[[gene-expression-matrix]]` links and gives retrieval a stable target.
