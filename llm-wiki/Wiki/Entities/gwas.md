# GWAS

> GWAS is a statistical method for associating genetic variants with traits. Canonical page is under Methods.

A Genome-Wide Association Study (GWAS) is a large-scale statistical study that tests associations between millions of [[snp|SNPs]] (single nucleotide polymorphisms) and a trait or disease outcome across many individuals.

Key concepts from Lecture 17:
- Tests each SNP for association with the phenotype (usually using logistic or linear regression)
- Results in a **Manhattan plot** showing âˆ’logâ‚â‚€(p-value) across the genome
- Must correct for [[linkage-disequilibrium|linkage disequilibrium (LD)]] between nearby variants
- GWAS hits feed into [[polygenic-risk-score|polygenic risk score (PRS)]] construction

**Why Methods, not Entities:** GWAS is a computational/statistical procedure applied to genomic data, not a biological entity.
