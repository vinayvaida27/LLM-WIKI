# SMILES

## Definition
Line notation for representing molecular structure using ASCII characters. The most common molecular string representation. Limitations include non-unique strings and occasional invalid chemistries.

## Why It Matters
SMILES is a canonical MLCB graph node. Keeping it as a source-backed Wiki page prevents Obsidian from creating a phantom node for `[[smiles]]` links and gives retrieval a stable target.
